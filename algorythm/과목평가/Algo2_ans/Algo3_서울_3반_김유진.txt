if ( ( i == 0 ) && ( j == 0)))

1) 빈 스택을 선언하고, 입력을 스트링으로 받는다.
2) 스트링에서 괄호를 제외한 부분은 무시하고, 스택에는 괄호만 취급한다.
3) 스택이 비어있을 때, 만약 닫는 괄호 ')'가 등장한다면 비정상적인 입력으로 검사를 종료한다.
4) 여는 괄호 '('가 등장하면 스택에 push한다.
5) 닫는 괄호 ')'가 등장하면 스택의 top이 여는 괄호 '('인지 확인한 후, 맞다면 스택에서 pop한다. 아니라면 비정상적인 입력으로 검사를 종료한다.
6) 이 과정을 반복한 후, 마지막에 스택이 비어있는지 확인하고, 비어있다면 정상적인 입력으로 검사가 종료되고, 비어있지 않다면 비정상적인 입력이다.

[ ] # 닫는 괄호를 만나 pop

-> 즉, 여는 괄호의 수와 닫는 괄호의 수가 일치하지 않다는 것
